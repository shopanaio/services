# Gift With Purchase (GWP)

Подарок при выполнении условия (по товарам / количеству / сумме). Gift-SKU показывается и становится доступен только при выполнении триггера.

## UI редактирования

### Основные поля

- **Условие активации** — select + поля:
  - Товар в корзине (product picker)
  - Количество товаров ≥ N (number input)
  - Сумма ≥ X (number input)
  - Товар из категории (category picker)
- **Подарок:**
  - Товар-подарок (product picker)
  - Количество подарков (number input, default 1)
  - Цена подарка — radio: Бесплатно | Символическая цена (number input)
- **Лимит** — max подарков на заказ (number input, default 1)
