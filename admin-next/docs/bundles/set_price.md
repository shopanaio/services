# Set Price (N for X)

Фикс "N за X" (например, 3 за 999) — вариант BYOB с фиксированной общей ценой. Покупатель выбирает N товаров из пула, платит фиксированную сумму.

## UI редактирования

### Основные поля

- **Количество товаров (N)** — number input
- **Цена набора (X)** — number input
- **Пул товаров** — multi-picker (из каких товаров можно выбирать)
- **Группировка** (опционально) — разделение пула на категории
