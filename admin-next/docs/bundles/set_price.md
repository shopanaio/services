# Set Price (N for X)

Фикс "N за X" (например, 3 за 999) — вариант BYOB с фиксированной общей ценой. Покупатель выбирает N товаров из пула, платит фиксированную сумму.

## UI редактирования

### Основные поля

- **Количество товаров (N)** — number input
- **Цена набора (X)** — number input
- **Пул товаров** — multi-picker (из каких товаров можно выбирать)
- **Группировка** (опционально) — разделение пула на категории

### Отличия от generic rule editor

- Ещё проще чем BYOB — **два числовых поля** (N и X) + пул товаров
- Нет min/max диапазонов — количество строго фиксировано
- Цена всегда фиксированная — нет вариантов скидки
- Один condition (qty = N) → один action (price = X)

### Генерируемые правила (dependency-rules)

```
Constraint:
  action: SET_QTY_LIMITS на GROUP (min: N, max: N)
  action: SET_REQUIRED(true) на GROUP

Валидация:
  condition: GROUP_TOTAL_QTY(pool) LT N
  action: DISABLE на BUNDLE

Ценообразование:
  condition: GROUP_TOTAL_QTY(pool) EQ N
  action: OVERRIDE_PRICE на BUNDLE (priceType: FIXED, priceValue: X)
```
