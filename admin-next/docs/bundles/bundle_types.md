# Product-level (бандл = продукт)

Покупатель взаимодействует с бандлом как с самостоятельной сущностью — страница/конфигуратор, бандл существует до попадания в корзину.

Fixed Bundle (Prebuilt set)
Жёсткий комплект: A+B+C → одна цена/скидка.

Multipack / Pack Size (Same-SKU pack)
Упаковка из N одинаковых единиц как отдельный pack-оффер (часто с отдельным SKU или автоматической группировкой).

Build-Your-Own (BYOB / Mix & Match)
Пользователь выбирает товары из пула, чтобы собрать набор (Pick N / Spend X).

# Cart-level (промо-правило на корзину)

Активируется автоматически на основе состояния корзины (что положили, сколько, на какую сумму).

Quantity Breaks (Volume / Tiered pricing)
Скидка/цена по ступеням количества: 1 / 2 / 3+.

Set Price (N for X)
Фикс "N за X" (3 за 999), независимо от цены отдельных позиций.

Buy X Get Y (BXGY / BOGO)
Купи X — получи Y (free / % off / fixed off).

Gift With Purchase (GWP)
Подарок при условии (по товарам/кол-ву/сумме). Отличается от BXGY тем, что "gift SKU" обычно отдельный сценарий (auto-add + limit).

Threshold Discount (Spend/Buy threshold)
Если выполнен порог (сумма/кол-во/категория) → скидка на корзину или на часть корзины.

Add-On / Attach Offer
Скидка на "прикреплённые" товары только вместе с триггер-товаром (upsell аксессуара).
