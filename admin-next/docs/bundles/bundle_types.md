# Типы бандлов

## Обзор

Shopana поддерживает несколько типов бандлов (комплектов), каждый из которых решает определённые задачи — от продажи готовых наборов до стимулирования оптовых закупок.

---

## 1. Классический бандл (Classic Bundle)

**Фиксированный набор товаров со скидкой при покупке всех позиций вместе.**

| Параметр       | Описание                                            |
| -------------- | --------------------------------------------------- |
| Структура      | Заранее определённый набор продуктов/вариантов      |
| Условие скидки | Покупатель добавляет в корзину все товары набора    |
| Типы скидок    | Процент, фиксированная сумма или фиксированная цена |

### Пример использования

Магазин косметики предлагает «Набор для ухода за лицом»: очищающий гель + увлажняющий крем. При покупке обоих товаров вместе клиент получает скидку 10%.

### Настройка

- Определить точный список продуктов в бандле
- Выбрать тип и размер скидки
- Опционально включить бесплатную доставку

---

## 2. Смешивай и сочетай (Mix & Match Bundle)

**Покупатель сам выбирает товары из предложенного списка, чтобы собрать комплект.**

| Параметр            | Описание                                          |
| ------------------- | ------------------------------------------------- |
| Структура           | Покупатель выбирает N товаров из допустимого пула |
| Условие скидки      | Выполнены условия выбора (мин. количество и т.д.) |
| Обязательные товары | Некоторые позиции можно пометить как обязательные |
| Типы скидок         | Процент, фиксированная сумма                      |

### Пример использования

Магазин одежды: «Выбери любые 3–5 футболок из коллекции и получи скидку 15%». Клиент может взять три одинаковые или пять разных — скидка применится автоматически.

### Настройка

- Определить пул допустимых товаров
- Задать мин./макс. количество для выбора
- Пометить определённые товары как `required` (скидка не сработает без них)
- Выбрать тип и размер скидки

---

## 3. Скидка за объём (Volume Discount)

**Многоуровневые скидки за покупку большого количества.**

| Параметр       | Описание                                        |
| -------------- | ----------------------------------------------- |
| Структура      | Один товар или товары из определённой категории |
| Условие скидки | Порог количества или суммы корзины              |
| Типы уровней   | Фиксированное количество, диапазон, минимум     |

### Пример использования

«Купи 5 товаров — скидка 10%, купи 10 — скидка 20%». Или по сумме: «Скидка 10% при сумме корзины свыше $200».

### Типы уровней

| Тип                  | Описание                       | Пример         |
| -------------------- | ------------------------------ | -------------- |
| Фиксированное кол-во | Скидка при точном количестве   | Каждые 5 штук  |
| Диапазон             | Скидка в пределах диапазона    | 2–5 единиц     |
| Минимальный порог    | Скидка при превышении минимума | Более 5 единиц |

### Настройка

- Определить подходящие товары/категории
- Задать условия уровней (по количеству или сумме)
- Назначить размер скидки для каждого уровня

---

## 4. BOGO (Купи один — получи второй)

**Купи обязательный товар — получи другой со скидкой.**

| Параметр       | Описание                                           |
| -------------- | -------------------------------------------------- |
| Структура      | Обязательный товар(ы) + товар(ы) со скидкой        |
| Условие скидки | Обязательный товар находится в корзине             |
| Типы скидок    | Процент (до 100% = бесплатно), фиксированная сумма |

### Пример использования

Магазин аксессуаров: «Купите кожаный ремень (обязательный товар) и получите второй такой же или любой из списка бесплатно (скидка 100%)».

### Настройка

- Определить обязательные (триггерные) товары
- Определить товары со скидкой (награда)
- Задать тип и размер скидки для товаров-наград

---

## Общие функции (все типы)

### Уровень продукта vs варианта

| Режим            | Поведение                                                              |
| ---------------- | ---------------------------------------------------------------------- |
| Уровень продукта | Товар отображается с выпадающим списком размеров/цветов                |
| Уровень варианта | Каждый вариант (напр. Красный S, Синий M) — отдельная позиция в бандле |

### Приоритетность

Если товар входит в несколько бандлов, приоритет определяет, какая скидка применяется первой. Более приоритетными обычно ставят бандлы с большим количеством товаров.

### Бесплатная доставка

Любой тип бандла может включать бесплатную доставку как дополнительный стимул поверх денежной скидки.
