# Fixed Bundle (Prebuilt Set)

Жёсткий комплект: A+B+C → одна цена/скидка. Покупатель не выбирает — берёт готовый набор целиком.

## UI редактирования

### Основные поля

- **Название бандла** — текстовое поле
- **Список товаров** — таблица/список с колонками:
  - Товар (picker)
  - Количество (number input, default 1)
  - Удалить
- **Ценообразование** — radio/select:
  - Фиксированная цена бандла (number input)
  - Скидка от суммы товаров (% или фикс)

### Генерируемые правила (dependency-rules)

```
Для каждой группы:
  action: SET_REQUIRED(true)

Для каждого item:
  action: SET_QTY(N)          — фикс количество
  action: DISABLE             — нельзя менять выбор

Ценообразование:
  action: OVERRIDE_PRICE на BUNDLE (priceType: FIXED, priceValue: X)
  // или
  action: ADJUST_PRICE на BUNDLE (priceType: PERCENT, priceValue: -N%)
```
