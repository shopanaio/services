# Multipack / Pack Size (Same-SKU Pack)

Упаковка из N одинаковых единиц как отдельный pack-оффер. Один SKU, фиксированное количество, своя цена.

## UI редактирования

### Основные поля

- **Товар** — product picker (один SKU)
- **Размер упаковки** — number input (N штук)
- **Цена упаковки** — radio/select:
  - Фиксированная цена пака (number input)
  - Скидка за единицу (% или фикс)
  - Цена за единицу в паке (number input)

### Отличия от generic rule editor

Максимально упрощённый UI — **три поля**:

- Нет groups/pools — всегда один товар
- Нет condition builder
- Количество = размер пака, не диапазон
- Цена задаётся напрямую

### Генерируемые правила (dependency-rules)

```
action: SET_QTY(N)                — фикс количество = размер пака
action: SET_QTY_LIMITS(min: N, max: N)  — нельзя менять
action: DISABLE                   — нельзя убрать из набора

Ценообразование:
  action: OVERRIDE_PRICE на ITEM (priceType: FIXED, priceValue: packPrice / N)
  // или
  action: ADJUST_PRICE на ITEM (priceType: PERCENT, priceValue: -X%)
```
