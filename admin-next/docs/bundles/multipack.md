# Multipack / Pack Size (Same-SKU Pack)

Упаковка из N одинаковых единиц как отдельный pack-оффер. Один SKU, фиксированное количество, своя цена.

## UI редактирования

### Основные поля

- **Товар** — product picker (один SKU)
- **Размер упаковки** — number input (N штук)
- **Цена упаковки** — radio/select:
  - Фиксированная цена пака (number input)
  - Скидка за единицу (% или фикс)
  - Цена за единицу в паке (number input)
